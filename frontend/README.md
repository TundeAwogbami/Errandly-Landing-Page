# Errandly
A courier service platform.

## Installation

### Prerequisites:
- nodejs
- vite
- react

### Installation:
1. Clone the repository: 
```bash
git clone https://github.com/TundeAwogbami/Errandly-Landing-Page.git
```

2. Install dependencies:
```bash
cd errandly
npm install
```

3. Run the project:
```bash
npm run dev
```
