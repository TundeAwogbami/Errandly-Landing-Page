# Errandly

A courier service platform.

## Installation

### Prerequisites:

- nodejs
- vite
- react

### Installation:

1. Clone the repository:

```bash
git clone https://github.com/TundeAwogbami/Errandly-Landing-Page.git
```

2. Install dependencies:

```bash
cd frontend
npm install
```

3.  Set up environment variables like so:

```bash
VITE_API_URL=your-api-url
```

4. Run the project:

```bash
npm run dev
```
